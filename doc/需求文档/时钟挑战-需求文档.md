# 儿童教学小游戏 - 时钟挑战

> **版本**：V1.0  
> **创建日期**：2026-06-18  
> **目标用户**：5~6 岁儿童

---

## 一、项目概述

### 1.1 项目背景

本项目是一款面向 5~6 岁儿童的网页教学小游戏，旨在通过趣味性的游戏玩法，帮助孩子学习小学基础知识（语文、数学、英语）。游戏采用 HTML5 技术开发，可直接在浏览器中运行，无需安装任何插件。

### 1.2 项目定位

- **产品类型**：儿童教育类网页游戏
- **目标用户**：5~6 岁学龄前儿童
- **核心价值**：寓教于乐，在游戏中学习时间认知

### 1.3 后续扩展

本项目采用模块化架构设计，主界面作为游戏入口平台，支持无限扩展游戏数量。每开发一个新的教学游戏，即可在主界面新增一个入口按钮。

## 四、时钟挑战 - 游戏详情

### 4.1 游戏简介

时钟挑战是一款帮助儿童学习认识时钟的教学游戏。通过两个不同的游戏模式，让孩子学会看时钟，分清楚时针和分针两个指针。

### 4.2 游戏目标

- 认识时钟的基本结构（表盘、12 个数字、分钟刻度线）
- 理解时针（红色短针）和分针（蓝色长针）的不同作用
- 学会读取时钟显示的时间
- 能够根据给定时间调整时钟指针的位置

### 4.3 其他按钮

| 按钮 | 说明 |
|------|------|
| 更多玩法 | 参考截图底部的圆角按钮，点击后可展开更多功能或跳转到其他游戏入口（可选功能） |

## 五、模式一：填时间模式（填一填）

### 5.1 游戏规则

1. 屏幕上显示一个时钟，时钟显示特定时间（参考截图样式）
2. 屏幕下方显示时间填空框，格式为：`[]时[]分`（简化版，不含秒）
3. 屏幕下方区域显示 0~9 的数字键盘
4. 玩家点击空位选中后，再点击数字键盘上的数字填入
5. 填写正确后，播放成功动画，点击"换一换"进入下一题
6. 点击"答案"按钮可显示正确答案

## 六、模式二：摆时钟模式（拨一拨）

### 6.1 游戏规则

1. 屏幕上方显示目标时间，如 "7:30"
2. 屏幕中央显示一个可交互的时钟，参考截图样式
3. 玩家通过拖拽红色时针和蓝色分针调整指针位置
4. 当时钟指针位置与目标时间匹配时，自动判定通关
5. 点击"答案"按钮可显示正确指针位置
6. 点击"换一换"按钮切换到下一道题目

### 6.2 界面布局

```
│                             │
│         目标时间             │
│           7 : 30            │
│                             │
│         ┌───────┐           │
│        /  12   /            │
│       │       │              │
│    9 ─│───────│─ 3          │
│       │       │              │
│        \  6   \              │
│         └───────┘            │
│           时钟               │
│                             │
├─────────────────────────────┤
│                             │
│   游戏任务: 将时针与分针     │
│   拨到指定时间               │
│                             │
│   方法: 先拨红色时针到大     │
│   致位置, 再拨蓝色分针微调   │
│                             │
└─────────────────────────────┘
```

### 6.3 时钟组件规格（可拖拽）

| 元素 | 规格 |
|------|------|
| 时钟直径 | 约 420px（与填时间模式一致） |
| 表盘颜色 | 纯白色背景 (#FFFFFF) |
| 表盘边框 | 黑色细边框，约 2px |
| 数字样式 | 12 个数字，蓝色 (#1E90FF)，字体大小适中，粗体 |
| 分钟刻度 | 60 个黑色小刻度线 |
| 时针 | 红色 (#F44336)，短针，长度 = 半径 × 0.55，宽度 8px，可拖拽，选中时高亮 |
| 分针 | 蓝色 (#2196F3)，长针，长度 = 半径 × 0.75，宽度 6px，可拖拽，选中时高亮 |
| 秒针 | 不显示（简化版） |
| 中心点 | 灰色 (#888888) 圆形，半径约 8px，固定不可拖拽 |
| 拖拽方式 | 点击指针末端附近区域并拖动 |

### 6.4 拖拽交互

| 操作 | 行为 |
|------|------|
| 点击/触摸红色时针 | 时针高亮（加宽或发光效果），可单独拖拽旋转 |
| 点击/触摸蓝色分针 | 分针高亮（加宽或发光效果），可单独拖拽旋转 |
| 拖拽方向 | 顺时针或逆时针旋转 |
| 角度步进 | 分针：6°/格（360°/60 分钟），时针：6°/格（跟随分针微调） |
| 实时反馈 | 拖拽过程中实时更新指针角度，无需松手即可看到效果 |
| 吸附效果 | 分针自动吸附到最近的分钟刻度 |

### 6.5 题目生成规则

| 难度等级 | 小时范围 | 分钟范围 | 说明 |
|----------|----------|----------|------|
| 简单 | 1~12 | 整 5 分（0, 5, 10, 15, 20, 25, 30, 35, 40, 45, 50, 55） | 适合初学者 |
| 中等 | 1~12 | 任意分钟（0~59） | 要求精确到分钟 |
| 困难 | 1~12 | 任意分钟（0~59） | 增加时间限制 |

### 6.6 操作方法说明（参考截图底部文字）

**游戏任务**：将时针与分针拨到指定时间

**操作方法**：先拨红色时针到大致位置，再拨蓝色分针微调，二者配合使用，完成速度最快。

**文字样式**："红色时针"和"蓝色分针"关键词用红色/蓝色标注，其他文字黑色

### 6.7 通关判定

```
判定间隔：每 500ms 检查一次
判定条件（仅检查时针和分针角度）：
  - 时针角度 = 目标时针角度 (±5° 容差，因时针可能被分针位置微调)
  - 分针角度 = 目标分针角度 (±3° 容差)

判定公式：
  - 时针角度 = (小时 % 12) × 30° + 分钟 × 0.5°
  - 分针角度 = 分钟 × 6°
```

### 6.8 通关条件

- 连续答对 5 题
- 记录用时和调整次数
- 显示通关动画和奖励

## 十、异常处理

### 10.1 网络异常

- 游戏无需网络连接，纯离线运行
- 无需处理网络异常情况

### 10.2 数据异常

| 场景 | 处理方式 |
|------|----------|
| LocalStorage 不可用 | 提示用户开启浏览器隐私设置 |
| 数据格式错误 | 清除错误数据，使用默认数据 |
| 数据版本不匹配 | 迁移或重置数据 |

### 10.3 边界情况

| 场景 | 处理方式 |
|------|----------|
| 时针指向 12 和 1 之间 | 按 12 计算 |
| 拖拽超出表盘范围 | 限制在表盘内 |
| 屏幕尺寸过小 (<320px) | 最小缩放至 0.5 倍 |

> **版本**：V1.0  
> **创建日期**：2026-06-18  
> **风格定位**：可爱糖果风，适合 5~6 岁儿童

## 一、风格概述

### 1.1 设计理念

本项目采用**糖果风格（Candy Style）**作为主要视觉风格，灵感来源于《糖果传奇》（Candy Crush）等经典消除类游戏。整体设计追求：

- **可爱甜美**：圆润的形状、温暖的色彩
- **立体质感**：3D 厚度感、高光阴影
- **活泼动感**：弹性动画、粒子效果
- **友好亲和**：保护视力、适合儿童

### 1.2 核心视觉元素

| 元素 | 说明 |
|------|------|
| 圆角矩形 | 所有元素使用圆角（16px~24px） |
| 渐变色 | 135° 方向的三色渐变 |
| 高光效果 | 顶部白色渐变高光 |
| 阴影效果 | 底部多层阴影营造厚度 |
| 粒子动画 | 点击消除时的飞散效果 |

## 四、渐变规范

### 4.1 糖果渐变（135° 方向）

所有糖果元素使用统一的 135° 渐变方向：

```css
background: linear-gradient(
    135deg, 
    var(--light-color) 0%,    /* 浅色 - 左上 */
    var(--main-color) 50%,    /* 主色 - 中间 */
    var(--dark-color) 100%    /* 深色 - 右下 */
);
```

**渐变示意图**：

```
    浅色
   ↙
  ↙
浅色 → 主色 → 深色
           ↘
            ↘
             深色
         
         ↖ 135° ↘
```

### 4.2 高光渐变

用于糖果顶部的光泽效果：

```css
.candy-block::after {
    background: linear-gradient(
        180deg,
        rgba(255,255,255,0.7) 0%,      /* 纯白 */
        rgba(255,255,255,0.3) 50%,     /* 半透明白 */
        rgba(255,255,255,0) 100%       /* 完全透明 */
    );
}
```

## 六、动画规范

### 6.1 悬停动画

鼠标悬停时的弹性效果：

```css
@keyframes bounce {
    0%, 100% { transform: scale(1); }
    50% { transform: scale(1.1); }
}

.candy-block:hover {
    animation: bounce 0.3s ease;
}
```

| 参数 | 值 | 说明 |
|------|-----|------|
| 动画名称 | bounce | 弹跳效果 |
| 持续时间 | 0.3s | 适中速度 |
| 缓动函数 | ease | 自然缓动 |
| 缩放范围 | 1 → 1.1 → 1 | 放大 10% |

### 6.2 消除动画（POP 效果）

糖果被消除时的爆炸效果：

```css
@keyframes pop {
    0% { 
        transform: scale(1); 
        opacity: 1; 
    }
    50% { 
        transform: scale(1.3); 
        opacity: 0.8; 
    }
    100% { 
        transform: scale(0); 
        opacity: 0; 
    }
}
```

**动画阶段**：

| 阶段 | 缩放 | 透明度 | 时长占比 |
|------|------|--------|----------|
| 开始 | 1 | 1 | 0% |
| 峰值 | 1.3 | 0.8 | 50% |
| 结束 | 0 | 0 | 100% |

### 6.3 粒子动画

点击消除时飞散的小粒子：

```css
@keyframes particle-fly {
    0% {
        transform: translate(0, 0) scale(1);
        opacity: 1;
    }
    100% {
        transform: translate(var(--tx), var(--ty)) scale(0);
        opacity: 0;
    }
}
```

**粒子效果参数**：

| 参数 | 值 | 说明 |
|------|-----|------|
| 粒子数量 | 8 个 | 均匀分布 |
| 飞行距离 | 60~100px | 随机 |
| 粒子大小 | 8px | 圆形 |
| 动画时长 | 0.5s | 快速消散 |

## 十、使用示例

### 10.1 创建红色糖果

```html
<div class="candy-block red" data-color="red">
    <div class="highlight"></div>
</div>
```

```css
.candy-block.red {
    --light-color: #FF6B7A;
    --main-color: #E8364F;
    --dark-color: #C0392B;
}
```

### 10.2 创建粒子效果

```javascript
function createParticles(x, y, color) {
    const particleCount = 8;
    
    for (let i = 0; i < particleCount; i++) {
        const particle = document.createElement('div');
        particle.className = 'particle';
        particle.style.backgroundColor = color;
        particle.style.left = x + 'px';
        particle.style.top = y + 'px';
        
        // 随机方向
        const angle = (Math.PI * 2 * i) / particleCount;
        const distance = 60 + Math.random() * 40;
        const tx = Math.cos(angle) * distance;
        const ty = Math.sin(angle) * distance;
        
        particle.style.setProperty('--tx', tx + 'px');
        particle.style.setProperty('--ty', ty + 'px');
        particle.style.animation = 'particle-fly 0.5s ease forwards';
        
        particlesContainer.appendChild(particle);
        
        // 清理粒子
        setTimeout(() => {
            particle.remove();
        }, 500);
    }
}
```

## 十一、注意事项

### 11.1 儿童友好设计要点

| 要点 | 说明 |
|------|------|
| 颜色对比度 | 确保文字与背景有足够对比度 |
| 点击区域 | 按钮和可点击元素至少 44px × 44px |
| 动画时长 | 动画时长控制在 0.3s~0.5s，避免过快或过慢 |
| 音效配合 | 重要操作配合音效反馈 |
| 无闪烁 | 避免任何闪烁效果，保护儿童视力 |

### 11.2 性能优化

- 使用 `transform` 和 `opacity` 实现动画（GPU 加速）
- 避免动画中改变布局属性（如 width、height）
- 粒子效果使用完毕后及时清理 DOM
- 使用 `will-change` 优化动画性能
